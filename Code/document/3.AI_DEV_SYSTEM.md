# 🤖 AI DEV SYSTEM — NỘI THẤT TIỆN LỢI

## 🎯 Mục tiêu
Hệ thống này giúp Claude Code hoạt động như một Senior Developer:
- Hiểu project trước khi code
- Tuân thủ architecture & convention
- Code đúng phạm vi
- Tự review & test

---

# 📦 1. Cấu trúc hệ thống

```
/project-root
├── skill-noithat-dev.ts        # Brain (context project)
├── skill-noithat-dev.md        # Guide nhanh
├── unit-test-checklist.md      # Checklist test
├── AI_DEV_SYSTEM.md            # File này
├── tasks/
│   ├── TASK_TEMPLATE.ts
│   ├── xxx_task.ts
```

---

# 🧠 2. Nguyên tắc cốt lõi

- KHÔNG code ngay
- PHẢI đọc project trước
- PHẢI có plan + risk
- CHỈ sửa file liên quan
- KHÔNG đoán khi thiếu context

---

# 🚀 3. Flow chuẩn

## Bước 1 — Analyze

```bash
Please read:
- ./skill-noithat-dev.ts
- ./unit-test-checklist.md
- ./tasks/your_task.ts

Then:
1. Analyze project structure
2. Identify related files
3. Summarize logic
4. Show plan
5. Show risks
DO NOT WRITE CODE
```

## Bước 2 — Confirm

```bash
Proceed with implementation.
Follow the skill strictly.
```

## Bước 3 — Code

- Theo đúng pattern
- Không sửa lan
- Giải thích file sửa

## Bước 4 — Test

```bash
Verify:
- edge cases
- follow checklist
- list test cases
```

## Bước 5 — Review

```bash
Review:
- architecture
- naming
- bugs
- unnecessary changes
```

---

# ⚙️ 4. Mode sử dụng

## Debug

```bash
Analyze:
- bugs
- missing logic
DO NOT CODE
```

## Feature

```bash
Implement task:
- follow skill
- minimal changes
```

## Safe

```bash
Only modify related files.
No global changes.
```
